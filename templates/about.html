About Our Project

Sentia is a unified sentiment analysis platform that leverages the power of natural language processing and computer vision to identify and interpret emotions in different types of content. It offers users the ability to analyze sentiments from raw text, visual images, and movie reviews scraped from IMDb. With a simple and interactive web interface, Sentia transforms complex data into understandable emotional insights, making it useful for personal analysis, academic learning, or business intelligence.

Text Analysis

Users can input any textual data such as comments, tweets, or feedback. The system uses natural language processing (NLP) models powered by TextBlob and VADER to determine whether the text expresses a positive, negative, or neutral sentiment. It's a fast and effective way to gauge public opinion or individual emotional tone in a text format.


Image Analysis

Sentia also supports sentiment analysis on images using deep learning models built with TensorFlow/Keras. Movie Review Analysis

Sentia allows users to analyze IMDb movie reviews through a simple input form. It scans the review using NLP and classifies it into a sentiment category, helping users understand general audience perception or conduct sentiment-based research on films. This functionality is particularly useful for entertainment insights or feedback studies.
